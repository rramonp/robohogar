## Patterns of the Top 10%

### 1. Un solo acento de color llamativo
Todos los top performers tienen UN color de acento que los hace reconocibles: amarillo (#e4f909), azul (#0a4e99), teal (#0ea890). ROBOHOGAR → verde lima `#C8FF00`.

### 2. Tipografía serif para headings = editorial authority
Los 2 mejores (Not Boring, we all are robots) usan Lora serif para títulos. Transmite seriedad editorial. ROBOHOGAR usa Syne (sans-serif bold) que transmite moderno/tech — funciona para nuestro vibe.

### 3. Content-first: enseña antes de pedir
Not Boring y we all are robots muestran contenido real antes del signup. Los peores (Robot Report) esconden el contenido detrás de un CTA genérico.

### 4. Identidad tribal > número de suscriptores
EVwire: "EV geeks". we all are robots: "robotics leaders". No dicen "Join 13K subscribers" — dicen "Join the tribe". ROBOHOGAR → "los que ya viven con robots".

### 5. Stats de calidad como trust signal
Weekly Robotics publica open rate (51%). Not Boring muestra engagement per post (343 reactions). Números de calidad > números de cantidad.
