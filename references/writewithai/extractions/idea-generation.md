### Patrones de voz notables

- 4 sub-topics dentro de cada A (Tips/Tools/Hacks/Advice etc.)

### Takeaway accionable
**Prompt completo literal** (uno de los más valiosos del newsletter):
```
I am going to train you to become an Endless Idea Generation Machine.
4 primary types: Actionable / Analytical / Aspirational / Anthropological
Sub-topics within each:
Actionable: Tips, Tools, Hacks, Advice, Resources, Frameworks, Ultimate Guides, Curation
Analytical: Trends, Numbers, Reasons, Examples, Teardowns, Swipe files
Aspirational: Lessons, Mistakes, Reflections, Personal stories, Stories of growth, Underrated traits, Advice to past self
Anthropological: Fears, Failures, Struggles, Paradoxes, Observations, Comparisons, Why others are wrong, Why you've been misled

I give Topic + FOR WHO + SO THAT. You generate 1 idea (as headline) per sub-topic.
```

### Aplicabilidad ROBOHOGAR
**Máxima prioridad.** Copiar este prompt literal al content-draft skill de ROBOHOGAR. Cada vez que Rafael esté sin ideas, ejecutarlo con:
- Topic: robótica doméstica española
- FOR WHO: adultos 30-55 en España
- SO THAT: elegir robot y disfrutar casa

Output esperado: 30+ ideas organizadas en 4A buckets, listas para usar en newsletter/social/artículo.

## 3. Unstick Your Brain With This 3rd Grade Technique (ABC Brainstorming)

- **URL:** https://writewithai.substack.com/p/unstick-your-brain-with-this-3rd
- **Likes / Fecha:** 88 / 2024-11-17
- **Aplicabilidad:** 🟢 directo. Artículo con contenido completo antes del último prompt.

### Estructura (scraped casi completa)
1. Hero + H1 + H3 "ABC Brainstorming"
2. "Hey there"
3. ABC Method: 26 ideas en 15 minutos, A-Z
4. Pasos: timer 15 min + write A-Z words relacionadas a topic
5. Ejemplo con "coffee" (A=Arabica, B=Brewing, C=Caffeine...)
6. Hacks para letras difíciles (Q, X, Y, Z)
7. H1 "The Alphabetizing Prompt" — prompt literal
8. Ejemplo output para coffee (A-Z lista completa scrapeada)
9. H2 "Expand An Idea" — 4 preguntas para cada letra (mistake, quick win, story, how-to)
10. [PAYWALL — prompt de expansion corta]

### Patrones de voz notables
- "3rd grade technique" — accesibilidad framing
- Hacks específicos para tricky letters — útil
- "I rarely make it all the way to Z—by the time I hit M, my mind is already swimming with ideas"

### Takeaway accionable
**Prompt literal:**
```
Generate a detailed A-Z analysis for [TOPIC], where:
- Each letter must have one key concept/term
- Each entry follows format: [LETTER] = [CONCEPT]
- All entries must directly relate to [TOPIC]

Topic: [TOPIC]
Target Audience: [SPECIFY]
Purpose: [SPECIFY]
```

Para cada letra, 4 sub-preguntas de expansión: biggest mistake / quick win / story / how-to.

### Aplicabilidad ROBOHOGAR
Directo: cada trimestre correr ABC Method para "robótica doméstica" en español. Output típico:
- A = Autonomía, Aspiradora
- B = Base de carga, Batería
- C = Compatibilidad, Cámara
- ...
- Z = Zona (mapeo zonas)
Cada letra → 4 artículos posibles = banco de 100+ ideas.
