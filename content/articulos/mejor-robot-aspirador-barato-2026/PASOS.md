### Crop obligatorio post-generación a 1200×630

Receta canónica en `assets/branding/nano-banana-prompt-base.md § Dimensiones`. Pillow:

```python
from PIL import Image
TARGET_W, TARGET_H = 1200, 630
TARGET_RATIO = TARGET_W / TARGET_H
for v in [1, 2, 3]:
    p = f"content/articulos/mejor-robot-aspirador-barato-2026/assets/hero-mejor-robot-aspirador-barato-2026-v{v}.png"
    img = Image.open(p)
    w, h = img.size
    src_ratio = w / h
    if src_ratio > TARGET_RATIO:
        new_w = int(h * TARGET_RATIO); left = (w - new_w) // 2
        img = img.crop((left, 0, left + new_w, h))
    else:
        new_h = int(w / TARGET_RATIO); top = (h - new_h) // 2
        img = img.crop((0, top, w, top + new_h))
    img = img.resize((TARGET_W, TARGET_H), Image.LANCZOS)
    img.save(p, 'PNG', optimize=True)
    img.save(p.replace('.png','.webp'), 'WEBP', quality=85, method=6)
```
